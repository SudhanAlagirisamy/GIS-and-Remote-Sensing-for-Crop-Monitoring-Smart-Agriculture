🌱 Smart Crop Monitoring using GIS & Remote Sensing

The Smart Crop Monitoring System is an advanced web-based platform that leverages 🌍 Geographic Information Systems (GIS) and 🛰️ Remote Sensing to enable precision agriculture and smarter farming decisions. It integrates satellite imagery, geospatial analysis, and real-time data processing to monitor crop conditions, optimize irrigation, and improve productivity.

The system uses 🛰️ Sentinel-2 data to fetch high-resolution images of agricultural land. It computes 🌿 NDVI to evaluate vegetation health. These values are visualized as overlays on RGB images, highlighting healthy crops 🌾, stressed areas 🌱, and barren land 🪨. Users can provide location input through place names, coordinates, or live GPS 📡 data. The backend processes the selected region and generates both satellite visuals 🖼️ and NDVI outputs 🌈. It also delivers insights such as crop health, type, and irrigation advice 💧.

The application is built using ⚙️ Flask with a responsive UI using 🎨 HTML, CSS, and Tailwind. A secure 🔐 authentication system with SQLite ensures safe access. This project demonstrates real-world GIS and remote sensing integration for smart agriculture 🚜. It transforms raw data into actionable insights, supporting sustainable farming 🌍 and efficient resource use.

Future improvements include interactive maps 🗺️, NDVI time-series analysis 📊, AI-based crop prediction 🤖, and cloud deployment ☁️. 🌱 Smart Crop Monitoring using GIS & Remote Sensing
🚀 Overview

The Smart Crop Monitoring System is an innovative web-based platform that integrates 🌍 Geographic Information Systems (GIS) and 🛰️ Remote Sensing to support precision agriculture. This project leverages satellite imagery and data analytics to provide real-time insights into crop health, helping farmers and researchers make informed decisions.

By utilizing Sentinel-2 satellite data, the system processes geospatial information and transforms it into meaningful visualizations such as NDVI overlays, enabling efficient monitoring of vegetation and land conditions.
